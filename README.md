# Agentic AI Healthcare Assistant

An autonomous system designed to help individuals get reliable, explainable, and personalized healthcare information through natural conversations

## Features
- Answers symptom-related questions using verified data 
- Provides clear explanations in everyday language
- Suggests when to seek professional help
- Includes Retrieval-Augmented Generation (RAG) for sourced answers

## Folders
 - data/ : medical csv/json files
 - src/ : agents and backend
